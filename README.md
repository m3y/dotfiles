How to install
--------------

1. Clone

```
$ git clone https://github.com/m3y/kctx.git
```

2. Install(create symlink)

```
$ ./install
  Install the following files.
    .gitconfig
    .tmux.conf
    .zprofile
    .zshrc
    :

  OK? (y/N): y
```
