dotfiles
========

Overview
--------
My dotfiles

Environment
-----------

| - |application name|
|---------|---|
|**shell**|zsh|
|**editor**|neovim|
|**terminal multiplexer**|tmux|
|**terminal**|alacritty|
|**application launcher**|ulauncher|

How to install
--------------

1. Clone
```
$ git clone git@github.com:m3y/dotfiles.git
```

2. Install(create symlink)
```
$ ./install
  Install the following files.
    .gitconfig
    .gitconfig.local
    .tmux.conf
    .zprofile
    .zshrc
    :

  OK? (y/N): y
```
