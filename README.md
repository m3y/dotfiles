dotfiles
========

Overview
--------
My dotfiles

How to install
--------------

1. clone
```
$ git clone git@github.com:m3y/dotfiles.git
```

2. install
```
$ ./install
  Install the following files.
    .gitconfig
    .gitconfig.local
    .tmux.conf
    .zprofile
    .zshrc
    :

  OK? (y/N): y
```

Check for differences
---------------------

```
$ ./rcdiff
```
